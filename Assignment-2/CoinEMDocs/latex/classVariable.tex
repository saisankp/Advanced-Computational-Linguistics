\doxysubsection{Detailed Description}
represents aspects of a variable 

mainly a given variable has particular range of values that it can have There is a symbol table included in the variable which maps the strings that occur in files to represent these values to integers

eg. a coin toss variable might map \textquotesingle{}H\textquotesingle{} to 0, and \textquotesingle{}T\textquotesingle{} to 1
